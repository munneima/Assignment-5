Important FAQ:

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a structure that represents a webpage as a tree of objects, allowing programming languages like JavaScript to interact with and modify the content and layout of a webpage.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like:

  1. getElementById() - by ID
  2. getElementsByClassName() - by class
  3. getElementsByTagName() - by tag
  4. querySelector() - by CSS selector
What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element, instead of multiple listeners to individual child elements. It works by leveraging event bubbling, making it more efficient and easier to manage events, especially when elements are added or removed dynamically.

How do you manipulate an event's attributes and styles using the DOM?

To manipulate an event's attributes and styles using the DOM:
Attributes: Use setAttribute() or directly modify properties (e.g., element.id = "newId").
Styles: Use style.property (e.g., element.style.backgroundColor = "red") to change CSS properties.
